# 丰富Zotero功能第一步：在Zotero软件内安装“应用市场”

# Zotero软件的内置插件市场下载

[Github链接](https://github.com/syt2/zotero-addons)

## 下载步骤：

1. 点进上述[Github链接](https://github.com/syt2/zotero-addons)后，往下拉滚动条直到看到“Download the latest release xpi file”的句子（或者`ctrl+F`这句话也行），然后按说明操作。
2. Download the latest release xpi file
3. Install in Zotero `Tools工具` -> `Add-ons插件`，直接把下载下来的xpi文件拖到界面上。（To 苹果人：你们的工具栏在屏幕最顶端）
4.
