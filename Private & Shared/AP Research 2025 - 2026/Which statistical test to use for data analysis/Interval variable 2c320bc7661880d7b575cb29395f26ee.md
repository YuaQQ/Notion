# Interval variable

An **Interval variable** is a type of numerical (quantitative) data where the difference between two values is **meaningful and constant** (i.e., the intervals are equal), but it lacks a true, non-arbitrary zero point.

# **Characteristics:**

- **Equal Intervals:** The difference between 10 units and 20 units is the same as the difference between 30 units and 40 units.
- **Arbitrary Zero:** A value of zero does not indicate the complete absence of the measured property. Ratios are meaningless.

# **Example:**

- Temperature measured in Celsius or Fahrenheit. 20℃ is 10℃ warmer than 10℃, but 20℃ is not twice as hot as 10℃ (because 0℃ is just an arbitrary freezing point).
