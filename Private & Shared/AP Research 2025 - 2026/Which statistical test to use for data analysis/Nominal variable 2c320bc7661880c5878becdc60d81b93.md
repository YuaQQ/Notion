# Nominal variable

**Nominal data** (or a **nominal variable**) is a type of categorical data used purely for **labeling** or **naming** categories, without any inherent order or quantitative value. It represents qualitative differences.

# **Characteristics:**

- **Classification:** Data points are simply assigned to groups.
- **No Order:** The categories cannot be meaningfully ranked (e.g., it makes no sense to say "blue is greater than red").
- **Operations:** The only mathematical operation that applies is counting the frequency of occurrences in each category (i.e., mode).
- **Examples:** Hair color (Blonde, Brown, Black, Red), Marital Status (Single, Married, Divorced), or Gender (Male, Female, Non-binary).

# How many levels can nominal data have?

Nominal data can have **two or more levels (categories)**.

- If it has exactly two levels (e.g., Yes/No, On/Off, Male/Female), it is a special case called a **dichotomous** or **binary** variable.
- If it has more than two levels (e.g., states in the US, types of trees), it is a **polytomous** variable.
