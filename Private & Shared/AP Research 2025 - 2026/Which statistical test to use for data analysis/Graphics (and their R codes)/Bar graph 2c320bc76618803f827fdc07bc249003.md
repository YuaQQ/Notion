# R code for bar graph (example)

- Use the following .csv file for the example coding

[skink.csv](Bar%20graph/skink.csv)

- A good package for plotting graphs that meet the publication standards is ‘**ggplot2**’.
- To modify the error bars, definitions of the SE and SD functions are required. The summary functions are performed by the package ‘**dplyr**’.
- The '**tidyverse**' package includes both '**dplyr**' and '**ggplot2**’.

```r
# Import the data to R and check the structure:
skink <- read.table('skink.csv',header=T,sep=',')
str(skink)

# Load the package 'tidyverse' for both 'ggplot2' and 'dplyr' functions
install.packages("tidyverse") # Run only if you haven't installed it yet
library(tidyverse) # Load the the downloaded package

# attach the dataset
attach(skink)

# summarize the dataset using the 'dplyr' package functions in 'tidyverse'
skink.sum <- skink |>
  # Group the data by the factors you want to summarize (SITE and SEX)
  group_by(SITE, SEX) |>
  # Calculate the summary statistics for each group
  summarise(
    N = n(), # Count of observations
    MEAN = mean(MASS.g, na.rm = TRUE), # Mean mass (g)
    sd = sd(MASS.g, na.rm = TRUE), # Standard Deviation
    se = sd / sqrt(N), # Standard Error (sd / sqrt(N))
    # Calculate the upper and lower bounds for the Mean ± 2*SE interval
    Lower_CI = MEAN - 2 * se,
    Upper_CI = MEAN + 2 * se,
    # The .groups = 'drop' ensures the output is a standard dataframe
    .groups = 'drop' 
  )

# View the new summary table
print(skink.sum)

# Do the bar graph with the 'ggplot2' package functions in 'tidyverse'
ggplot(skink.sum, aes(x = SITE, y = MEAN, fill = SEX)) +
  
  # 1. Add the bar geometry for the mean
  # position_dodge ensures bars for different SEX are side-by-side
  geom_col(position = position_dodge(width = 0.9), color = "black") +
  
  # 2. Add the error bar geometry
  geom_errorbar(
    # Use the calculated 2*SE bounds for ymin and ymax
    aes(ymin = Lower_CI, ymax = Upper_CI),
    # Make error bars slightly narrower than the bars themselves
    width = 0.25, 
    # Use the same dodging width as geom_col
    position = position_dodge(width = 0.9) 
  ) +
  
  # 3. Apply labels and theme customization
  scale_y_continuous(limits = c(0, 35)) + # Set the y-axis limit as requested
  labs(
    y = "Skink Mass (g)", # y-axis label
    x = "Study Sites", # x-axis label
    fill = "Sex" # Title for the fill/grouping legend
  ) +
  
  # Optional: Customize the appearance (e.g., use a cleaner theme)
  theme_classic()

```
