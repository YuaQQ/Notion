# Normal Distribution

[Source Page](https://www.researchoptimus.com/article/normal-binomial-poisson-distribution.php)

# Characteristics:

- Occurs naturally in numerous situations.
- Data points are similar and occur within a small range.
- Much fewer outliers on the low and high ends of data range

# Example:

- To standardize a Normal Distribution (convert a raw data value ‘X’ to a standard score ‘Z’)：

![image.png](Normal%20Distribution/image.png)

- x = Raw data value that is being standardized
- μ = Mean of the distribution
- σ = Standard deviation of the distribution

- Assuming a specific population has μ = 4, and σ = 2. The Z score corresponding to X = 6 will be:

![image.png](Normal%20Distribution/image%201.png)

- Z = 1 means that the value of X = 6 which is 1 standard deviation above the mean.
