# One-way ANOVA

# of predictors: 1
Example Research Question: Does swallow mass differ across different SITEs?
Factor Type: Between-Subjects

[Back to ANOVA](../../ANOVA%202c320bc7661880989015e2e06dd78bf6.md)

# General characteristics

- Relationship between variables:
    - **y ~ A**
- Independent variable:
    - 1 categorical independent variable (between-subjects factor)
- Dependent variable:
    - 1 [continuous](../../../Continuous%20variable%202c320bc7661880b9a90ff829067c71f3.md) (quantitative) variable

# Hypotheses

- H0: There are no differences among the mean DVs for the ≥3 experimental populations.
- Ha is not directional.

# Analysis of variance (ANOVA) F test

- The test of H0 against Ha

![image.png](One-way%20ANOVA/image.png)

- Large values of F are evidence against the null hypothesis
- If p< 0.05, H0 is rejected (The population means are not equal)

# Statistics required

- Experimental group names
- Sample sizes
- Sample means
- Sample variances (SDs)

# Example

- IV: SITE (2, 4, 9)
- DV: MASS

```r
# ====================== 1. 安装并加载必要的包 ======================
# 如果还没安装，先取消下面一行的注释来安装
# install.packages(c("tidyverse", "car", "multcomp", "rstatix", "ggpubr"))

library(tidyverse)   # 数据处理和绘图
library(car)         # Levene检验
library(multcomp)    # 多重比较
library(rstatix)     # 提供一些简便的统计函数
library(ggpubr)      # 作图辅助

# ====================== 2. 准备数据 ======================
# 假设你的数据框名为 data，其中：
#   var1 是数值型因变量（例如测量值）
#   var2 是因子型自变量（分组变量，例如处理组）
# 确保 var2 是因子类型
data <- data %>%
  mutate(var2 = as.factor(var2))

# 查看数据结构
glimpse(data)

# ====================== 3. 拟合 ANOVA 模型 ======================
# 使用 aov() 函数，结果保存为 data.aov
data.aov <- aov(var1 ~ var2, data = data)

# 查看传统的 ANOVA 表
summary(data.aov)

# ====================== 4. 检验模型假设 ======================
# 假设1：残差独立性（通过实验设计保证，例如随机分组）
# 假设2：残差正态性（可用 Shapiro-Wilk 检验）
shapiro.test(residuals(data.aov))
# 若 p > 0.05，则满足正态性假设

# 假设3：方差齐性（Levene检验，car包）
leveneTest(var1 ~ var2, data = data)
# 若 p > 0.05，则满足方差齐性

# 也可以画 Q-Q 图和残差 vs 拟合图来直观诊断
par(mfrow = c(2, 2))
plot(data.aov)  # 基础诊断图
par(mfrow = c(1, 1))

# ====================== 5. 事后检验（如果ANOVA显著） ======================
# 如果 ANOVA 的 p 值 < 0.05，可以进行多重比较
# 常用的 Tukey HSD 检验
tukey_result <- TukeyHSD(data.aov)
print(tukey_result)

# 用 multcomp 包进行更灵活的多重比较（例如与对照组比较）
# 假设 var2 的第一个水平为对照组
dunnett <- glht(data.aov, linfct = mcp(var2 = "Dunnett"))
summary(dunnett)
```
