# Nested ANOVA

# of predictors: >2
Example Research Question: Does the variation in mass among individual BIRDS (Nested) differ within each SITE?
Factor Type: Between-Subjects

[Back to ANOVA](../../ANOVA%202c320bc7661880989015e2e06dd78bf6.md)

# **Goal:**

- Account for the fact that one factor (e.g., **Treatment**) is administered to a higher-level unit (e.g., a **Field Plot**), and the lower-level units (e.g., individual **Plants** or **Birds**) within that unit are the true replicates. This prevents *pseudoreplication*.

# **Design**

- General formula
    - **Y ~ A/B/C**
- One factor is **nested** within the levels of another factor.
- **Data Structure Needed:** Requires an ID for the higher-level unit.
