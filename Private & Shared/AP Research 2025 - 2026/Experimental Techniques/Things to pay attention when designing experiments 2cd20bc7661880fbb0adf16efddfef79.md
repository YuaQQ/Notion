# Classes of data & the datatype in [R](../Introduction%20to%20R%20(Cr%20Amber%20Zhang)%202bf20bc7661880d8a42efd8e53fc0077.md)

| **Class** | **Notes** |
| --- | --- |
| **Continuous** | A continuous number
In R continuous data is **Numeric (Num)** |
| **Discrete** | A set of categories. Also called categorical or factorial
In R a category is a **Factor** |
| **Binomial** | Yes/No data.
In R binomial data is a **Factor** |
| **Ordinal** | Numerically ranked data.
In R ordinal data will be an **Integer (Int)** |
| **Count** | Whole numbers that will never be negative
In R count data will be an **Integer (Int)** |
