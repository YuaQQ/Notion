# Transformation + Agar Plate Culture

# 所用试剂盒

生工（B528413）

产品组成：

1. E.coli DH5α Competent Cells (10 x 100ul, -80℃)
2. pUC19 plasmid (0.1 ng/ul, 10ul, -80℃)

# 自备材料

- 1.5ml离心管
- DNA样品（非pUC19扩增情况下）
- 冰盒
- 移液枪
- 枪头
- 废液缸
- 离心管架
- 42℃水浴
- 秒表
- 漂浮海绵架
- 摇床
- 37℃培养箱
- 无菌SOC培养基
- LB液体培养基
- 酒精灯
- 打火机
- 消毒湿巾

# 步骤

1. 桌面消毒，手套消毒。
2. 从-80℃冰箱内取出*E.coli* DH5α Competent Cells，迅速插入冰盒内，使其溶解。
3. 加入DNA样品轻轻混匀，冰上放置30分钟。
4. 42℃水浴热激45秒，迅速放回冰上放置2分钟左右，注意不要晃动。添加700ul不含抗生素的无菌培养基，混合均匀。
5. 37℃振荡培养1小时（160-225 rpm），吸取适量体积均匀涂布到含相应抗生素的LB琼脂培养基平板中。37℃正置至液体被吸收，然后倒置过夜培养。
