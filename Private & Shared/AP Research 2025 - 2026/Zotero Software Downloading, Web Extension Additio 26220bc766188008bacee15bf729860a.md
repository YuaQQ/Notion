# Zotero Software Downloading, Web Extension Addition, and Recommended Plugins

### Zotero 7软件下载与安装

[Zotero下载地址](https://www.zotero.org/)

下载后，默认安装路径在C:\Program Files\Zotero，不喜欢C盘莫名其妙塞满东西的人可以手动改路径到别的盘。安装好的Zotero应该会在Word里自动安装插件，使Word工具栏中出现Zotero选项卡。

- 如果下载Zotero后Word插件没有直接安装，手动下载并安装步骤如下：
1. [手动下载Zotera.dotm链接](https://www.zotero.org/support/word_processor_plugin_manual_installation)。下载地址属于Github，最好挂梯子。
2. [安装教程](https://www.cnblogs.com/erebus-qie/p/16184426.html)

### Zotero软件的内置插件市场下载

[Github链接](https://github.com/syt2/zotero-addons)

下载步骤：

1. Download the latest release xpi file
2. Install in Zotero (Tools) -> (Add-ons)，直接把下载下来的xpi文件拖到界面上

使用步骤：

1. After installing this add-on in Zotero, click

![image.png](Zotero%20Software%20Downloading,%20Web%20Extension%20Additio/image.png)

in Toolbar, or click `Add-on market` in `Tools` menu.

![image.png](Zotero%20Software%20Downloading,%20Web%20Extension%20Additio/image%201.png)

然后爱下啥下啥！

### Zotero常见插件

（[Zotero中文社区](https://zotero-chinese.com/)是个好地方！）

1. [**Better Notes**](https://github.com/windingwind/zotero-better-notes)

拿来做笔记的。支持笔记模板系统，像obsidian一样做双链笔记，自动根据你的笔记和链接生成可视化的知识图谱，提供markdown支持可以输出到obsidian或notion中查看，建立知识体系。

1. [**Jasminum](https://github.com/l0o0/jasminum)（茉莉花）**

为 Zotero 内置的 PDF 阅读器添加高级批注功能。

1. [**Sci-PDF**](https://github.com/syt2/zotero-scipdf)

自动从sci-hub下载文献。

1. [**Green Frog**](https://github.com/redleafnew/zotero-updateifsE)

显示文献影响因子。

1. [**Ethereal Reference**](https://github.com/MuiseDestiny/zotero-reference)

阅读文献时在旁边显示参考文献。可以直接保存参考文献到文库，还可以按时间/被引用书对参考文献进行排序。（cred. 小红书@明天就早睡早起）
