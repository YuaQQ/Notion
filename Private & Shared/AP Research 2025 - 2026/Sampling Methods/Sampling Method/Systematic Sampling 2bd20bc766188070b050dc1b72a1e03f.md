# Systematic Sampling

Core Principle: Fixed interval random selection
Primary Applicable Scenarios: Randomly ordered list
Type: Probability

The findings from the sample need to be generalized to the population, aiming for representativeness and objectivity. Allows for statistical inference.

# Procedure

1. Population units are ordered
2. A random starting point is chosen,
3. Every k-th unit (sampling interval) is selected.

# Scenario

- The population list is randomly ordered.
- More convenient than simple random sampling.

# Example

Selecting every 10th student from an alphabetically ordered roster.
